## 設計

```
host:9999でnginx受け付けて、nginxからappサーバuwsgiに8080でproxyする。
uwsgiは8080で受けて、5000のflaskに接続する。
443はALBからしか使われないので、この領域はhttpになる。local以外のostは:80になる。
```
